

# 📦 Required Packages

To install all the necessary packages for this script, use the following commands:

```bash
pip install --upgrade pip
pip install -U telethon
pip install psutil
pip install asyncio
pip install aiocron
pip install pytz
pip install googletrans==4.0.0-rc1
pip install gtts
pip install requests
```
